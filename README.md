# Pizza Wednesday

_An oasis in the desert_

## Getting Started
- Ask to be a contributor
- `git clone https://github.com/byrnehollander/pizza-wednesday.git`
- `yarn install`
- `yarn start`
- To deploy to GitHub Pages, run `yarn run deploy`
